## 经典双重差分法（2×2 DID）

接下来我们就将正式进入双重差分法（Difference-in-Differences, DID）的世界。

这是一个**非常经典且非常优雅的因果推断方法**。和传统计量经济学那种“只要回归就能解释世界”的感觉不同，DID更像一个**精心设计的实验策略**：
它通过构造一个“如果政策没有发生，本来会怎样”的世界，来识别政策的真实影响。

如果你是第一次接触因果推断，可能会遇到很多新名词，比如：

* 潜在结果（Potential Outcomes）
* 反事实（Counterfactual）
* 平行趋势（Parallel Trends）
* ATT（Average Treatment Effect on the Treated）

不用担心，这些概念看起来很学术，但**本质都很直觉**。接下来我们一步一步把它拆开来看。

---

## Q1：为什么会有双重差分法（理论前提）？

这一部分可能有一点抽象，但我们尽量用通俗的语言解释。

## 1. 因果推断的核心问题

假设政府推出了一项政策，比如：

> **某些城市开始实行环保限排政策**

我们想知道：

**这项政策是否真的降低了污染？**

如果你是计量经济学初学者，可能第一反应是：

> 看看政策后的污染是不是下降了。

但问题来了。

假设某个城市污染下降了，我们怎么知道：

* 是政策导致的
  还是
* 本来污染就会下降？

这就是因果推断的核心难题。

---

## 2. 潜在结果框架（Potential Outcomes）

在因果推断中，我们用一个非常重要的思想：**潜在结果框架**。

对于个体 $i$，有两个潜在世界：

* $Y_i(1)$ ：如果接受处理（treatment）时的结果
* $Y_i(0)$ ：如果没有接受处理时的结果

比如：

| 情况       | 含义           |
| -------- | ------------ |
| $Y_i(1)$ | 城市实施环保政策后的污染 |
| $Y_i(0)$ | 如果没有政策时的污染   |

我们真正想知道的是：

$$
\boxed{\text{因果效应} = Y_{it}(1) - Y_{it}(0)}
$$

这就是：

**同一个体，在同一时间，处理与不处理的差异。**

但问题是：

> **这个量永远无法直接观察。**

因为现实世界里：

一个城市不可能同时

* 既实施政策
* 又不实施政策

这就是著名的 **Fundamental Problem of Causal Inference**。

---

## 3. 为什么要关注 ATT？

由于无法观察两个世界，我们只能退而求其次：

研究平均处理效应。

特别是：

**被处理群体的平均处理效应**

即：

$$
ATT = \mathbb E[Y_{i,t=2}(1)|D_i=1] - \mathbb E[Y_{i,t=2}(0)|D_i=1]
$$

这里的含义是：

| 项   | 含义               |
| --- | ---------------- |
| 第一项 | 处理组在政策后的真实结果     |
| 第二项 | 如果处理组没有被处理，本来会怎样 |

注意第二项：

> **这是一个反事实（counterfactual）。**

我们在现实世界中无法直接观察它。

---

## 一个更直观的例子

假设有两个城市：

| 城市 | 是否实施政策 |
| -- | ------ |
| A  | 实施     |
| B  | 未实施    |

政策实施后：

| 城市 | 污染变化 |
| -- | ---- |
| A  | -10  |
| B  | -5   |

A 城市污染下降 10。

但如果没有政策呢？

也许它本来就会下降 5（和 B 一样）。

那么真正的政策效果其实是：

```
-10 - (-5) = -5
```

这就是 DID 的核心思想。

---

## 辨析：$Y_{it}$ 和 $Y_{it}(0)$ 的关系

很多初学者都会困惑：

> $Y_{it}$ 和 $Y_{it}(0)$ 到底有什么区别？

其实很简单：

* $Y_{it}$ 是我们**真实观测到的数据**
* $Y_{it}(0)$ 是**如果没有政策时会发生什么**

形式化表达为：

$$
Y_{it} = D_{it}Y_{it}(1) + (1-D_{it})Y_{it}(0)
$$

含义是：

| 情况    | 实际观测        |
| ----- | ----------- |
| $D=1$ | $Y_{it}(1)$ |
| $D=0$ | $Y_{it}(0)$ |

换句话说：

我们永远只能看到两个潜在结果中的一个。

---

## Q2：要满足什么假设？

双重差分并不是魔法，它成立需要一些关键假设。

## 1. 平行趋势假设（Parallel Trends）

这是 **DID最核心的假设**。

含义是：

> 如果没有政策，处理组和控制组的变化趋势应该是一样的。

换句话说：

两组可能水平不同，但**趋势必须一样**。

直观理解：

```
污染
│
│   处理组
│    /
│   /
│  /
│ /
│/________ 时间
 控制组
```

如果政策没有发生，两条线应该继续平行。

一旦政策发生：

处理组偏离了趋势。

这部分偏离就是政策效果。

---

## 2. SUTVA 假设

SUTVA = Stable Unit Treatment Value Assumption

简单说就是：

**一个人的处理不会影响别人。**

例如：

如果我们研究补习班效果：

* 学生A补习
* 不应该影响学生B的成绩

否则就存在 **溢出效应（spillover）**。

---

## 3. 无预期效应

含义是：

> 在政策真正实施之前，个体不会提前改变行为。

否则就会出现：

**政策还没开始，效果已经出现。**

这会破坏 DID 的识别。

---

## Q3：双重差分的推导

我们从 ATT 开始：

$$
ATT = \mathbb E[Y_{i,t=2}(1)|D_i=1] - \mathbb E[Y_{i,t=2}(0)|D_i=1]
$$

第一项可以直接观测：

$$
\mathbb E[Y_{i,t=2}|D_i=1]
$$

问题仍然是：

**第二项是反事实。**

---

## 利用平行趋势假设

平行趋势假设告诉我们：

如果没有处理：

处理组和控制组的变化应该一样。

$$
\mathbb E[Y_{i,t=2}(0)|D_i=1] - \mathbb E[Y_{i,t=1}(0)|D_i=1]
=============================================================

\mathbb E[Y_{i,t=2}(0)|D_i=0] - \mathbb E[Y_{i,t=1}(0)|D_i=0]
$$

整理得到：

$$
\mathbb E[Y_{i,t=2}(0)|D_i=1]
=============================

\mathbb E[Y_{i,t=1}(0)|D_i=1]
+
\mathbb E[Y_{i,t=2}(0)|D_i=0]
-----------------------------

\mathbb E[Y_{i,t=1}(0)|D_i=0]
$$

进一步利用 SUTVA 和无预期假设，可以将潜在结果替换为观测值。

最终得到经典的 DID 公式：

$$
ATT =
(\mathbb E[Y_{i,t=2}|D_i=1] - \mathbb E[Y_{i,t=1}|D_i=1])
---------------------------------------------------------

(\mathbb E[Y_{i,t=2}|D_i=0] - \mathbb E[Y_{i,t=1}|D_i=0])
$$

---

## 为什么叫“双重差分”？

因为我们做了 **两次差分**：

第一步：时间差分

```
处理组：  后 - 前
控制组：  后 - 前
```

第二步：组间差分

```
(处理组变化) - (控制组变化)
```

所以叫：

**Difference-in-Differences**

---

## Q4：用什么办法估计 ATT？

在实践中，我们通常有两种方法。

## 方法1：简单粗暴法

直接算四个均值：

| 组   | 时间     |
| --- | ------ |
| 处理组 | before |
| 处理组 | after  |
| 控制组 | before |
| 控制组 | after  |

然后套公式。

---

## 方法2：回归法（TWFE）

更常见的方法是使用回归：

$$
Y_{it} = \alpha + \beta D_{it} + \gamma_i + \delta_t + \epsilon_{it}
$$

其中：

| 项          | 含义     |
| ---------- | ------ |
| $\gamma_i$ | 个体固定效应 |
| $\delta_t$ | 时间固定效应 |

在 **2×2 DID** 中：

```
β = ATT
```

---

## Q5：如何处理干预前协变量？

现实数据中，处理组和控制组往往**并不完全可比**。

例如：

| 城市  | GDP   |
| --- | ----- |
| 处理组 | 20000 |
| 控制组 | 8000  |

这种情况下：

我们希望控制组更像处理组。

一种方法是：

**加权（weighting）**

目标是找到权重 $w_i$：

$$
\sum_{i \in control} w_i X_i = \bar X_T
$$

意思是：

让控制组在协变量上**看起来像处理组**。

这类方法包括：

* 合成控制（Synthetic Control）
* SDID
* Matrix Completion
* IFE

---

## 最后总结

双重差分的核心思想其实非常简单：

**用控制组来构造处理组的反事实。**

换句话说：

> 如果没有政策，处理组会像控制组一样变化。

于是：

```
政策效果
=
处理组变化
-
控制组变化
```

这就是：

**Difference-in-Differences**

看起来只是两次简单的减法，却成为了现代政策评估中最重要的工具之一。
